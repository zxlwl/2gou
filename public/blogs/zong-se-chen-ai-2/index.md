# ✨<font style="color:#58a401;">核心玩法</font>
## 1. 核心战斗：3x4 战棋回合制（策略核心）

战斗在 3x4 网格棋盘进行，玩家操控最多 5 名角色（工会战 7 人），核心是 “站位 + 行动顺序 + 技能连携” 的战术组合，拒绝纯数值碾压。

## 2.  角色养成：多维度成长体系（战力核心）

角色养成是长线核心，围绕 “等级、觉醒、技能、装备、服装” 展开，资源需合理分配。

## 3. PvE 玩法：主线 + 日常 + 挑战（资源核心）

   PvE 是资源获取与剧情推进的核心，涵盖 “主线冒险、日常副本、高难挑战” 三类。

## 4. 特色玩法：生活技能与休闲内容（趣味延伸）

  游戏包含钓鱼、料理、派遣猫咪等生活技能，不仅趣味十足，还能产出养成资源。

-钓鱼：多人同船，按现实时间分昼夜时段，蓄力抛竿，鱼类技能增加挑战，升级船只提升收益。

-料理制作：用食材制作料理，提供战斗增益，食材可通过探索与钓鱼获取。

# 💽<font style="color:#58a401;">游戏安装</font>
---

## <font style="color:#c21c13;">第一步：解压游戏文件</font>
游戏下载完成后文件夹内是10个待解压缩的文件运行`棕色尘埃2.exe`进行解压缩

```plain
棕色尘埃2 文件目录结构
├── 棕色尘埃2.exe
├── 棕色尘埃2-1.bin
├── 棕色尘埃2-2.bin
├── 棕色尘埃2-3.bin
├── 棕色尘埃2-4.bin
├── 棕色尘埃2-5.bin
├── 棕色尘埃2-6.bin
├── 棕色尘埃2-7.bin
├── 棕色尘埃2-8.bin
└── 棕色尘埃2-9.bin
//**EXE与BIN后缀文件一共10个，少一个都无法解压**//
```

## <font style="color:#c21c13;">第二步：开启服务端</font>
![](/blogs/zong-se-chen-ai-2/a5e14a7ee14b9c95.png)

服务端路径：`棕色尘埃2` --> `2.Neowiz` --> `服务端` --> `运行`Bd2.Server.Api.exe`

> ⚠️<font style="color:#c21c13;">图片中最后出现的黑色窗口千万不要关掉</font>

## <font style="color:#c21c13;">第三步：启动游戏</font>
游戏客户端路径：`棕色尘埃2` --> `2.Neowiz` --> `Browndust2` --> 运行`BrownDust II.exe`

进入游戏后，等待更新，随便输入用户名与密码就可以进入游戏了
